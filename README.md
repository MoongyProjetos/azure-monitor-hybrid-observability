# Azure Monitor – Hybrid Observability

Este repositório contém os **materiais práticos, exemplos e laboratórios** da formação
**Azure Monitor para Ambientes Híbridos (Azure + On-Prem)**.

O objetivo é apoiar a implementação de uma **estratégia de observabilidade completa**
utilizando o **Azure Monitor**, cobrindo:

- Monitorização de serviços Azure (App Services, Azure SQL, Storage Accounts)
- Observabilidade de aplicações com **Application Insights**
- Monitorização de servidores on-premises com **Azure Arc**
- Criação de **Data Collection Rules (DCR)**
- Queries **KQL** para análise e alertas
- Dashboards e **Workbooks operacionais**
- Alerting e integrações (Teams, Email, Jira)
- Governance, controlo de custos e boas práticas empresariais
